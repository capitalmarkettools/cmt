The Purpose

We are under active development and the work is part of a finance and software development research initiative. The ultimate purpose is to provide an easy to use but rigorous financial engine for the retail as well as the professional investor. The development and architecture uses the most up to date software tools that are on the market.

Main Features

  • Only open source tools and data providers used
  • All application development is done with dynamically typed scripting language with web framework - Python and Django
  • Rigourous financial analytics engine which is used by Wall Street and Hedge Funds - QuantLib
  • Complete set of security definitions and market data
